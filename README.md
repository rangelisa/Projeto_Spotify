# 🎧 Projeto Spotify Clone

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **interface inspirada no Spotify**, criada utilizando **React**, com o objetivo de praticar conceitos de desenvolvimento Front-End, componentização e construção de interfaces modernas.

A aplicação simula funcionalidades visuais de uma plataforma de streaming de música, focando na experiência do usuário e organização de componentes.

---

## 🚀 Funcionalidades

✅ Interface inspirada no Spotify
✅ Layout moderno e responsivo
✅ Navegação entre páginas
✅ Componentes reutilizáveis
✅ Estrutura organizada em React
✅ Simulação de playlists e músicas

## ▶️ Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/seu-usuario/Projeto_Spotify.git
```

### 2️⃣ Acessar a pasta do projeto

```bash
cd Projeto_Spotify/client
```

### 3️⃣ Instalar dependências

```bash
npm install
```

### 4️⃣ Executar aplicação

```bash
npm start
```

A aplicação estará disponível em:

```
http://localhost:3000
```
